# Facial Attendance System

## Overview

A facial attendance system developed using Python and OpenCV that can register faces, train face data, recognize faces in real time, and automatically record attendance with date and time.

## Features

* Face Registration
* Face Training
* Real-Time Face Recognition
* Automatic Attendance Logging
* CSV-Based Attendance Records

## Technologies Used

* Python
* OpenCV
* CSV
* Pickle

## Project Structure

facial-attendance-system/

├── faces/

├── register.py

├── train.py

├── attendance.py

├── attendance.csv

└── face_data.pkl

## How to Run

### Register a Student

python register.py

### Train the Model

python train.py

### Start Attendance

python attendance.py
